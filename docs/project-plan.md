# Project Plan

## MVP

The first version of the app should allow a user to:

- create a workout
- add exercises to a workout
- log sets with reps and weight
- view workout history

## Out of Scope for MVP

- authentication
- advanced analytics
- calorie estimation
- polished charts
- workout templates

## Sprint Plan

- Sprint 0: setup, project structure, planning
- Sprint 1: backend Express API foundation
- Sprint 2: PostgreSQL schema and persistence
- Sprint 3: React dashboard and form flows
- Sprint 4: history, filtering, and progress views
- Sprint 5: stretch features and cleanup
