# Workout Tracker

## Goal

Build a full stack workout tracker that allows users to create workouts, add
exercises, log sets, reps, and weight, and review workout history over time.

## Tech Stack

- React
- Node.js
- Express.js
- PostgreSQL

## Project Structure

- `client/` - frontend dashboard
- `server/` - backend API
- `docs/` - planning notes and schema drafts

## Core Features

- Create workouts
- Add exercises
- Track reps and weight
- View workout history

## Stretch Features

- PR tracking
- Workout templates
- Progress charts
- Calories burned estimation
